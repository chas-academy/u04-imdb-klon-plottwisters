## Getting started
- Clone the project
- ```composer install```
- ```npm install```
- create .env file with envexample ```cp .env.example .env```
- ```php artisan key:generate```
- ```npm run build``` (if error run ```npm install vite@6.0.8 --save-dev```)
- Go to .devcontainer folder and run ```docker compose up```
